# Bubble.kiwi website

A static site as part of our entry into GovHack2016

## Installation

### Dependencies

- Ruby, v2.3.1+
- NodeJS, v6+

### Installation steps

```
bundle install
npm install
```

## Development

To start a local server (once the dependencies are installed) you'll need to:

```
middleman
```

Then browse to [http://localhost:4567](http://localhost:4567)
